How the services work

Making requests

To use our services you will need a web server to host your applications
No headers are required but you must be granted access before using the service
All requests must be posted to our base URL (provided below)

Response format and status codes

All service responses are JSON formatted documents with UTFÂ­8 encoded content
Only 200 (OK) status code exists, it might be delivered or not

Service types

Existing service version has 2 types: PUBLIC and PRIVATE
Both types are free and available to access
PUBLIC type is used for fetching brawler stats, gamemodes and maps
PRIVATE is used for application development

Base URL and enpoints

The base URL of our services is https://brawlout1.github.io/services/
PUBLIC services can be accessed by {base URL}/public and PRIVATE services can be accessed by {base URL}/private
Endpoints can be accessed by {base URL}/{service type}/{endpoint}
